What is a continuation?

Continuation (delimited continuation) is a program object representing a computation that may be suspended or resumed.

Implemented in JVM.

Implemented as a lower level construct.

Continuations:

  • Nested continuations
  • StackWalker
  • Forced preemption
  • Higher level continuations
  • Cloning
  • Serialization
  • Native monitors

Current limitations

  • cannot yield with native frames on continuation stack
  • cannot yield while holding monitor

What is a fiber?

Fiber is a light-weight (or user-mode) thread that is scheduled by the Java virtual machine, not by the operating system (kernel).

Fibers are intended to be very small footprint, so that we should be able to have millions of them with negligible task-switching overhead.

What is a fiber?

Fiber = continuation + scheduler
